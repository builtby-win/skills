# Note

Capture learnings, insights, or cool things from your current project as a draft blog post.

## What it does

1. Asks what you learned or want to document
2. Gathers relevant context (code snippets, explanations)
3. Creates a draft blog post at builtby.win/blog
4. Tags the post with the source project
5. Optionally commits the draft for later review

## When to use

- When you discover something interesting while coding
- When you solve a tricky problem worth sharing
- When you want to document a technique for future reference
- When you build something cool and want to write about it

## Usage

```bash
/note
```

The command will:
- Ask what you want to document
- Help you capture relevant code or context
- Generate a draft post with `draft: true`
- Save to your portfolio blog directory
